# Numerical Expression Calculator

## 개요
프로그램 실행 후 입력받는 수식을 통해 수식의 맞는 결과를 출력하는 일종의 계산기를 구현한다.

## 사용된 환경 & version
- Ubuntu 18.04.1 LTS (64bit)
- gcc (Ubuntu 7.4.0-1ubuntu1~18.04.1) 7.4.0
- flex 2.6.4

## 사용 시 주의사항
### Makefile에 정의 된 매크로를 사용하여 조금 더 편리하게 사용한다.

```bash
make
./a.out
1+2
```

**입력및실행**

```bash
make clean
```
